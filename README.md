# aetna-research-lab

Aetna's public research lab for trustworthy multimodal AI agents in real product workflows.

This repository is meant to grow over time. Articles may add small benchmarks, trace schemas, evaluators, fixtures, or research prototypes under `articles/`.

The code here is not a claim that an agent is safe. It is a public artifact trail for inspection, critique, and reuse.

## Current research line

How can an AI system read screens, infer user intent, take or propose actions, and show clear evidence for why it did them?

## Layout

- `articles/`: code artifacts attached to specific Aetna X Articles.

## Status

Research prototypes. Validate before production use.
